# SENVIROX — Smart Solutions for Environmental Recycling

Premium corporate website for SENVIROX — a technology company delivering digital solutions while building India's future in sustainable e-waste recycling.

## 🚀 Quick Start

### Option 1: Open Locally
Just open `index.html` in any web browser. No build step, no dependencies.

### Option 2: Host on GitHub Pages
1. Create a new repository on GitHub (e.g., `senvirox-website`)
2. Upload all files from this folder:
   - `index.html` (the complete website)
   - `.nojekyll` (tells GitHub Pages to skip Jekyll processing)
   - `README.md`
3. Go to **Settings → Pages**
4. Under **Source**, select **Deploy from a branch**
5. Select **main** branch and **/ (root)** folder
6. Click **Save**
7. Your site will be live at `https://yourusername.github.io/senvirox-website/` within 1-2 minutes

### Option 3: Host Anywhere
Upload `index.html` to any web hosting service:
- Netlify (drag & drop)
- Vercel
- Cloudflare Pages
- Any cPanel / shared hosting
- AWS S3
